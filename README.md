Design Decisions:

*Created a warm, elegant, and luxurious look using a refined palette of deep black, cream, and gold tones.
*Used Cormorant Garamond for headings and DM Sans for body text to achieve a sophisticated yet readable typography balance.
*Focused on subtle details like custom cursor, smooth scroll animations, gold hover effects, and a light noise texture to give a premium, artisanal feel.
*Kept the layout clean and logical: Hero → About → Services → Work → Testimonials → Contact.

Challenges Faced:

*Making the complex layout fully responsive, especially the hero and portfolio sections.
*Balancing rich visual effects with good performance.
*Building a smooth testimonials slider with autoplay and navigation.

Improvements for Future:

*Optimize images and add lazy loading for better speed.
*Add a working backend for the contact form.
*Enhance mobile experience and add light/dark mode toggle.
*Implement a better portfolio gallery with project modals.
